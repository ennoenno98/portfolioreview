## Decisions taken (user-confirmed via Q&A)

| Question | Decision |
|---|---|
| Review unit | **SKU × country** (each Amazon marketplace separate; webshop = one market `WEB`) |
| Release dates | **First-sale proxy** + editable `data/overrides/launch_dates_overrides.csv`; SKUs already selling Jul 2024 are *censored* |
| Margin anchor | **AP26 absolute targets** (gates) + **relative 3×3 margin × volume tiers** (priority) |
| Deliverable | **Streamlit app + committed monthly markdown pack** |
| Incubation | **6 months** post-launch, shielded from Harvest/Exit |
| Exit rule | Persistent negative CM3 (≥3 of last 6 months, window negative overall, not improving); **top-volume losers get one Fix cycle first** ("EXIT CANDIDATE") |
| Shopify margin gap | **Provisional CM from COGS** + placeholder fees in `config/shopify_assumptions.yaml` (user to fill real numbers) |

## What was built (repo layout)

```
app.py                        Streamlit dashboard, Vanatari CD 2026
pipeline/common.py            paths, constants, targets, helpers
pipeline/build_portfolio.py   sources -> data/generated/facts_monthly.csv.gz (+ launch dates, excel history, tags)
pipeline/classify.py          rules engine -> verdicts.csv + verdict_history/
reports/build_review_pack.py  monthly markdown pack -> reports/packs/review_YYYY-MM.md
config/rules.yaml             thresholds + verdict definitions (EDIT ME)
config/ap26_targets.json      plan targets (copied from productpricer/data/targets.json)
config/shopify_assumptions.yaml  webshop fee placeholders (EDIT ME)
data/source/                  input snapshots (see below)
data/overrides/               launch_dates_overrides.csv, verdict_overrides.csv
data/generated/               committed pipeline outputs
novadata_exports/             margin_export_2026-07-08.csv.gz (12m daily CM per SKU x marketplace)
```

Monthly cadence: refresh sources → `python pipeline/build_portfolio.py` →
`python pipeline/classify.py` → `python reports/build_review_pack.py` →
review in `streamlit run app.py` → overrides → commit.
